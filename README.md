This is the base repository for the `modernized-visions-saas-template`, bootstrapped with Next.js App Router, TypeScript, Tailwind CSS, and ESLint.

## Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Base Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment

Cloudflare/OpenNext deployment wiring is included in this template. Runtime variables, secrets, and bindings should still be managed manually in the Cloudflare dashboard whenever possible.
